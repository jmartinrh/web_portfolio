Portfolio

Courtyard King: The Walking Dead Universe (2025)

Platform: Fortnite Creative / UEFN

Roles: Project Lead, Game & Level Designer


An adaptation of The Walking Dead IP into a 3-way king of the hill-style game with roguelite elements in Fortnite Creative.

  • Led the project from concept to live-service launch, coordinating cross-discipline collaboration.
  • Designed and implemented core gameplay, progression systems, reward system, enemy waves, and metagame loops.
  • Designed player ranking system.
  • Built and greyboxed core level layouts and wayfinding flows to support gameplay, tone, and pacing

Peak Players: 3,891

Core Challenges

Designing a dynamic enemy spawn and waves system that could keep all game areas challenging while prioritizing big enemy hordes in the capture points with limited resources.


Making the interconnected level readable and navigable for all teams as capture points changed location.

Boss Lab Project (2024)

Platform: PC / UE 5.5

Roles: Project Lead, Game Designer


A boss-rush roguelike where creating more powerful boss-encounters defined character progression.

  • Designed modular skill and ability systems for both player and boss characters.
  • Desgined and implemented skill tree system and UI using UMG and BPs.
  • Designed and implemented player's range abilities using BPs.
  • Conducted playtesting and player surveying to refine controls and balance.

Core Challenges

Exploring a counterintuitive approach to player agency: advancing was about creating your own tougher challenges, not mechanical empowerment.


Designing modular character skills and abilities for enemies that could be mixed and matched during play

Species (2024)

Platform: Tabletop Card Game

Roles: Project Lead, Game Designer


Species is a 2-4 player card game where each takes the role of ecosystem stewards and work towards keeping their chosen ecosystem in balance as new species arrive. The objective is to foster a diverse ecosystem while managing effects triggered by species.

Download Game v1.0 Ruleset
    • Designed card mechanics and card layouts for an educational, environment-themed education game.
    • Conducted research on edutainment games and Canadian invasive species.
    • Organized and facilitated playtest sessions, gathering data on mechanics’ clarity, scientific content, layout, and user experience.

Core Challenges

Balancing accessibility, gameplay, and scientific accuracy to ensure the game was engaging and informative without overloading players with facts.


Making a game that could be a teaching tool but was not itself teaching the content. I wanted to foster a context where players could learn.

Forgotten Runiverse (2023)

Platform: PC / Web

Roles: RPG Game Designer


A fantasy pixel art MMORPG with TBA combat system where players play as wizards that can craft their own spells and items.

  • Defined combat and item formulas.
  • Designed character abilities and items.
  • Designed combat encounters and rewards.
  • Developed combat system simulations to inform balancing decisions.
  • Coordinated with other systems designers to balance player-led economy.

Core Challenges

Balancing stats, items, and combat that were intricately tied to a player-led creations and economy.


Creating engaging asymmetry in item combinations and stat bases to encourage player builds.
